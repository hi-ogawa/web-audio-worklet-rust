# WebAudio Worklet in Rust

simple example of integrating Rust into [`AudioWorkletProcessor`](https://developer.mozilla.org/en-US/docs/Web/API/AudioWorkletProcessor)

```sh
pnpm i
pnpm build-wasm
pnpm dev
```
